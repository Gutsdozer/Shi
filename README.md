# :globe_with_meridians: Python-Selenium-Automation project
This is a project for automation of basic functionality of the 'https://shishi.co.il/' web site using Python/Pycharm and Selenium. There are two different approaches that are used to make a test data: using object with prefilled fields and using Google Gemini for generating test data with given prompt.

## Functionality overview
* Registraion of a new user
* Logging in a newly registered user profile
* Logging out

## Used technology
* Python v3.12.7
* Selenium
* ChromeDriver
* pytest framework
* POM project architecture

## Setup
Use command line for all steps
* Clone git repository: "git clone https://github.com/Gutsdozer/Shi.git", then "cd Shi"
* Create a virtual environment:

for Windows: "python -m venv venv", then "venv\Scripts\activate"

for macOS/Linux: "python3 -m venv venv", then "source venv/bin/activate" 

* Install dependencies: "pip install -r requirements.txt"
* Be shure that you have ChromeDriver installed on your machine and it's path is signed in the PATH environment variable

## Using
Lanch tests using a command line
* Test data from User class object: "(venv) $ pytest tests/test_site.py::test_registration_user_object"
* Test data generated by Google Gemini: "(venv) $ pytest tests/test_site.py::test_registration_generated_test_data"

## Example of automated scenario
* Entering site from the homepage
* Entering register page
* Filling all necessery fields for a new user registration
* Clicking submit button
* Going to login page
* Filling login form with newly registered user's credentials
* Clicking login button and entering a new user profile
* Performing an exit from profile using exit menu
